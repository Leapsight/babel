

# Module babel_config_manager #
* [Description](#description)
* [Function Index](#index)
* [Function Details](#functions)

A server that takes care of initialising Babel configuration
with a set of statically define (and thus private) configuration options.

__Behaviours:__ [`gen_server`](gen_server.md).

<a name="description"></a>

## Description ##
All the logic is handled by the [`babel_config`](babel_config.md) helper module.<a name="index"></a>
