    get(Key, KVTerm)

    View Source 
    -spec get(Key :: key(), KVTerm :: t()) -> Value :: term().

    Returns value Value associated with Key if KVTerm contains Key. Key can be an atom, a binary or a path represented as a list of atoms and/ or binaries.

    The call fails with a {badarg, KVTerm} exception if KVTerm is not a property list, map or Riak CRDT Map. It also fails with a {badkey, Key} exception if no value is associated with Key.

    > In the case of Riak CRDT Maps a key MUST be a riakc_map:key().
